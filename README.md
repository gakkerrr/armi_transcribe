# armi_transcribe

Минимальный скрипт для транскрибации аудио/видео через Whisper.

После `git clone` нужен только один запуск команды:

`python app.py <input_file> <output_file>`

Пример:

`python app.py "input.mp4" "result.txt"`

## Что делает скрипт автоматически

- Устанавливает недостающие зависимости: `numpy`, `openai-whisper`, `imageio-ffmpeg`.
- Использует встроенный `ffmpeg` из `imageio-ffmpeg` (без ручной системной установки ffmpeg).
- Если входной файл видео, конвертирует его в mp3.
- Кэширует mp3 в `.audio_cache/` и при повторном запуске использует уже готовый файл.
- Автоматически выбирает модель Whisper по объему VRAM.

## Требования

- Python 3.9+
- Интернет при первом запуске (для установки пакетов и загрузки модели Whisper)
